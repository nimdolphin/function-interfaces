Function Interfaces

You can use these inputs to check the following functions:

DNA to RNA Conversion:

Enter a DNA sequence to convert it into an RNA sequence.
Only A, C, G, and T are allowed. For example, "GATTACA" converts to "CAUUUGU".

Palindrome Check:

Enter a string or number to check if it is a palindrome.
For example, entering "radar" or "404" will confirm that they are palindromes.

Substring Extraction:

Enter a string, start position, and length to extract a substring.
For example, for "JavaScript programming" with a start position of 0 and a length of 10, the result is "JavaScript".
